# QuantumTradeAI

AI-powered stock market analysis and trading platform.

## Features
- Real-time stock market analysis
- Technical indicators (RSI, MACD)
- AI-based trend prediction
- Paper trading
- Portfolio analytics

## Tech Stack
- Python (FastAPI)
- React / Next.js
- PostgreSQL
- Azure Cloud
- Machine Learning

## Project Structure

/backend → API and trading engine  
/frontend → UI dashboard  
/ai-models → ML models  
/docker → containerization  
/docs → documentation
